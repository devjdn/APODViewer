Information

APODViewer allows you to view information and images based off of NASA's Astronomy Picture of the Day API. This includes: Viewing today's APOD, using our date picker to view previous APODs, and using a number value selector to randomly generate between 4 and 100 of NASA's APODs.

I believe everyone should be able to have easy access to the array of incredible images of our universe that space agencies and independent astrophotographers take, and these simple API calls formatted into an easy to read layout was the way I thought would be best.

Built with love by Jayden Priestley.
